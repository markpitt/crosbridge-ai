Chrome Prompt API Bridge

Open this page in ChromeOS to connect the browser Prompt API to the local OpenAI-compatible endpoint.

Disconnected Prompt API unknown

How to use

  1. Start the server in Crostini with npm start or node src/server.js --no-open.
  2. Let the server open this page automatically, or paste the printed URL into ChromeOS.
  3. Send OpenAI-compatible requests to /v1/chat/completions.
curl http://127.0.0.1:8787/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer local" \
  -d '{
    "model": "chrome-prompt-api",
    "messages": [
      {"role": "system", "content": "You are concise."},
      {"role": "user", "content": "Explain the Django ORM in one paragraph."}
    ],
    "stream": true
  }'
